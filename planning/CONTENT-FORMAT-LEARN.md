# FourFlow LEARN Content Format Template

## Standard Structure for Educational Content Pieces

### Header Section
```
# [CONCEPT NAME]: [Descriptive Subtitle]
*A FourFlow Framework Exploration*

**Dimension**: [SELF/SPACE/SPIRIT/STORY]
**Key**: [Specific key from dimension]
**Type**: LEARN
**Duration**: [Est. reading time]
**Complexity**: [Beginner/Intermediate/Advanced]
```

### 1. Hook & Opening Context (100-150 words)
Start with a relatable question, scenario, or tension that immediately engages the reader. Connect to universal human experience.

**Structure:**
- Opening question or scenario
- Brief context of the challenge/opportunity
- Transition to the core concept

### 2. Core Concept Definition (150-200 words)
Clear, concise explanation of the main idea with practical grounding.

**Include:**
- Simple definition
- Why it matters
- How it fits within FourFlow framework
- Real-world relevance

### 3. Scientific Foundation (200-300 words)
Evidence-based backing for the concept.

**Include:**
- Relevant research studies (cite 2-3 key sources)
- Neurological/psychological mechanisms
- Data or statistics when available
- Connection to flow state science

### 4. Framework Integration (200-250 words)
Show how this concept connects to the broader FourFlow system.

**Structure:**
- Primary dimension connection
- Cross-dimensional relationships
- How it supports flow triggers
- Synergies with other keys

### 5. Practical Application (300-400 words)
Concrete strategies for implementation.

**Include:**
- 3-5 specific techniques
- Step-by-step instructions
- Common obstacles and solutions
- Success metrics

### 6. Real-World Example (150-200 words)
Case study or detailed scenario showing the concept in action.

**Include:**
- Specific context
- Application of the concept
- Results achieved
- Key lessons

### 7. Integration Reflection (100-150 words)
Questions and prompts to help readers integrate the learning.

**Include:**
- 3-4 reflection questions
- Action steps for next 24 hours
- Connection to personal mission/values

### 8. Further Exploration (50-100 words)
Resources and connections for deeper learning.

**Include:**
- Related FourFlow concepts
- Recommended books/articles
- Practical exercises

## Writing Guidelines

### Tone & Style
- Conversational yet authoritative
- Present tense, active voice
- Second person ("you") for engagement
- Scientific rigor with accessible language

### Length Requirements
- Total: 1,200-1,800 words
- Bite-sized for focused learning
- Scannable with clear subheadings

### Quality Standards
- Evidence-based claims
- Actionable insights
- Clear practical application
- Framework integration
- Flow state relevance

### Scientific Rigor
- Minimum 2 credible sources
- Current research (within 10 years preferred)
- Proper attribution
- Balanced perspective on limitations
