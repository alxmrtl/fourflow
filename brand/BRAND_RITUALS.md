# Brand Rituals

*How to apply FourFlow principles in daily practice*

---

## The Morning Orientation

Before creating anything for FourFlow, take 60 seconds:

1. **Ground** — One breath. Feel your body in the chair.

2. **Ask** — "What am I creating from?"
   - The Hermit (solitude, depth)?
   - The Fire (courage, initiation)?
   - The Awakener (transmission, service)?

3. **Clarify** — "Does this serve the frequency, or just fill space?"

Then begin.

## Pre-Creation Checklist

### Before Writing

- [ ] Have I read what I'm about to write out loud in my head?
- [ ] Am I writing to impress, or to help?
- [ ] Is there a single word from the banned list?
- [ ] Would I be slightly embarrassed showing this to someone I respect?
- [ ] Does this sound like me, or like an algorithm?

### Before Designing

- [ ] Could an AI generate this from a generic prompt?
- [ ] Is every element earning its place?
- [ ] Am I designing for timelessness or for trends?
- [ ] Does this feel like FourFlow, or like "wellness app"?
- [ ] Have I removed everything that doesn't need to be here?

### Before Shipping

- [ ] Does this carry the frequency?
- [ ] Is there texture, or is it glass-lands smooth?
- [ ] Would the hermit approve? The fire? The awakener?
- [ ] Am I shipping with soul, or shipping for metrics?
- [ ] Can I stand behind this in a year?

## The Decision Framework

When facing any choice, run it through the filters:

### Filter 1: The Frequency Test

*"Does this amplify the FourFlow frequency, or dilute it?"*

The frequency is: flow consciousness, authentic presence, spiritual pragmatism, community awakening, living systems with soul.

If a decision doesn't serve this, it doesn't matter how clever, efficient, or trendy it is.

### Filter 2: The Anti-Monument Test

*"Am I building a living system or a dead form?"*

Living systems:
- Evolve and iterate
- Breathe and respond
- Serve the community
- Stay humble about what they don't know

Dead forms:
- Claim permanence
- Resist feedback
- Serve the creator's ego
- Pretend to have all the answers

### Filter 3: The Hermit Test

*"Would I create this if no one was watching?"*

If the answer is no, examine why. Some work genuinely needs audience consideration. But work done purely for external validation rarely carries soul.

### Filter 4: The Fire Test

*"Am I waiting for perfection, or shipping with courage?"*

The struggle phase applies to creating, too. The first 10 minutes of writing/designing/building will feel hard. That's the on-ramp. Push through.

Imperfect and shipped beats perfect and imagined.

### Filter 5: The Awakening Test

*"Will this help someone enter flow today?"*

Not "will this impress?" Not "will this go viral?" Not "will this prove something?"

Will it help?

## Weekly Rituals

### Monday: Intention Setting

Ask: *What will I create this week that serves the frequency?*

Write it down. One to three things. No more.

### Wednesday: Mid-Week Calibration

Ask: *Am I still on the path I set Monday, or did I drift into glass-lands territory?*

Correct course if needed. It's not failure—it's navigation.

### Friday: Review

Ask:
- What did I create that carried soul?
- Where did I slip into cliché or formula?
- What did I learn about the brand by applying it?

Write brief notes. These become institutional memory.

Select three recent pieces of content/design/code. Score each on:

| Criterion | 1-5 |
|-----------|-----|
| Frequency alignment |   |
| Texture (not smooth) |   |
| Voice authenticity |   |
| Visual distinctiveness |   |
| Serves the user |   |

Average score guides next month's focus.

### Mid-Month: Cliché Purge

Review any recurring content (email templates, UI copy, marketing pages).

- Hunt for banned words and phrases
- Look for patterns that have become automatic
- Ask: "Is this still alive, or has it become formula?"

Update what's calcified.

## Quarterly Rituals

### Brand Soul Review

Read BRAND_SOUL.md in full.

Ask:
- Has anything shifted in my understanding of the archetype?
- Is the brand still expressing the same frequency?
- Have we drifted from any core principles?

Note observations. Evolve the documents if needed—but slowly, and with intention.

### Competitive Landscape Scan

Look at 5-10 other products in the flow/focus/wellness space.

Note:
- What are they all doing that we shouldn't?
- What clichés have emerged since last quarter?
- Where is there opportunity to be different?

Add any new anti-patterns to CLICHE_RADAR.md.

## Situational Rituals

### When You're Stuck

1. Step away from the screen
2. Ask: "What would the hermit do?"
3. Usually: go deeper, not wider. Simplify, don't add.

### When Something Feels Off

Trust the feeling. Then interrogate it:
- Is this too smooth/polished?
- Am I performing instead of expressing?
- Did I add something for external validation?
- Is the frequency intact?

### When You've Been Away

Before returning to FourFlow work after a break:
1. Reread BRAND_SOUL.md (the archetype grounding)
2. Reread a few examples from VOICE_EXAMPLES.md
3. Take one conscious breath
4. Begin with something small

### When Receiving Feedback

Ask:
- Does this feedback serve the frequency?
- Is the person responding to soul or surface?
- What can I learn without abandoning the vision?

Not all feedback is equal. Listen deeply, but don't be swayed by every opinion.

## The Content Pipeline

### For Writing (Articles, Emails, Social)

1. **Draft** — Write fast, don't edit. Get it out.
2. **Cool** — Walk away for at least 1 hour (ideally overnight).
3. **Audit** — Read aloud. Hunt banned words. Check voice.
4. **Polish** — Light editing only. Don't sand away texture.
5. **Ship** — Publish before it's perfect.

### For Design (UI, Graphics, Marketing)

1. **Sketch** — Low-fidelity exploration. Multiple directions.
2. **Select** — Choose the direction that carries the frequency.
3. **Refine** — Build out with intention. Every element earns its place.
4. **Test** — Does it work? Is it accessible? Does it feel right?
5. **Ship** — Launch knowing you'll iterate.

### For Code (Features, Fixes)

1. **Clarify** — What problem does this solve? How does it serve flow?
2. **Build** — Write with care. Unseen details matter.
3. **Test** — Does it work? Does it feel right?
4. **Review** — Check against brand principles (especially animation timing).
5. **Ship** — Deploy with confidence in the foundation.

## Emergency Interventions

### "This looks like every other wellness app"

Stop. Open VISUAL_MOODBOARD.md. Ask:
- What would Dieter Rams remove?
- What would feel like a contemplative space, not a marketplace?
- Where is the texture?

Redesign from first principles.

### "This sounds like ChatGPT wrote it"

Stop. Open CLICHE_RADAR.md. Hunt for:
- Banned words (delve, leverage, journey...)
- Banned phrases (In today's world, unlock your potential...)
- Three-adjective stacks
- Generic inspirational filler

Rewrite in your actual voice. Read it aloud.

### "I've lost the thread"

Stop everything. Return to the core question:

*"What would help someone enter flow right now?"*

Everything serves that. Nothing else matters.

This takes 30 seconds. It changes everything.
