# Single Metric Focus

> Before each session, identify ONE metric that will move if you work well — then track it visibly throughout the session.

## Mechanism
Dopamine fires in response to reward *prediction*, not just reward receipt. A visible counter incrementing toward a session goal provides continuous prediction signals throughout the session — creating a sustained motivational state, not a single post-session release. When the reward signal is abstract or delayed (eventual publication, eventual mastery), the dopamine loop can't close on a session-by-session basis. A single, concrete, session-relevant metric creates a local reward signal that functions regardless of long-term outcome.

## Protocol

1. Before starting the session, identify ONE metric that will move during this session if you work well. Choose the most direct measure of actual work, not quality or outcomes:
   - Writing: words drafted (not edited — editing is separate)
   - Code: tests passing (not features shipped)
   - Research: sources read and processed (not conclusions formed)
   - Design: iterations completed (not decisions made)
2. Write or display this metric target visibly — a whiteboard, sticky note, counter in your notes, or tab on screen.
3. Track the number as it moves. The act of watching it increment is the mechanism — each increment is a micro-reward signal.
4. Do not add a second metric mid-session. One metric per session.

## When to Use
At the start of any deep work session. The metric must be chosen *before* starting so the brain has a reward target from the first minute. Choosing mid-session loses the prospective dopamine effect.

## Key Insight
> "Effort that can't be measured feels like effort that doesn't count. One visible number closes the loop."
