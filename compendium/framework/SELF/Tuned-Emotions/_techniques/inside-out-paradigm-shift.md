# Inside-Out Paradigm Shift

> A reframe that moves agency from circumstances to internal patterns — transforming anxiety from "something happening to me" into "a pattern I can work with."

## Mechanism
Most people operate from an "outside-in" paradigm: circumstances determine emotional state, so they try to control circumstances to feel good. This is exhausting and ultimately futile because circumstances are never fully controllable. The inside-out shift recognizes that reactions are determined by internal patterns, not external events — the same circumstance triggers anxiety in one person and not another. This reframe is not just conceptual: it changes which question you ask, which determines the action you take.

## Protocol

1. When anxious, notice the default question you're asking: "How do I fix this situation?" or "Why is this happening to me?"
2. Replace it with the inside-out question: "What pattern is causing me to relate to this situation this way?"
3. Hold the question — don't rush an answer. The question itself creates a shift from reactivity to observation.
4. Act from whatever clarity emerges. The situation may still require action, but you act from insight rather than reactivity.

## When to Use
In any moment of anxiety where circumstances seem to be the source. Particularly useful as a first-response reframe before a full re-patterning session is possible. This technique doesn't transform the underlying pattern — it creates enough distance to stop the anxiety from running unexamined, and frames the pattern correctly for later re-patterning work.

**Contraindication**: Don't use this to dismiss legitimate external concerns. The inside-out paradigm doesn't mean circumstances are irrelevant — it means your emotional reaction is always about the pattern, even when action on the circumstance is also warranted.

## Key Insight
> "The same circumstance that triggers anxiety in you might not trigger someone else — the difference is the pattern, not the circumstance."
