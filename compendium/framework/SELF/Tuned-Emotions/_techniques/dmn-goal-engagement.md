# DMN Goal Engagement

> Give the brain a single specific goal to suppress the Default Mode Network's anxiety loop in real time.

## Protocol

1. When anxiety arises and a full re-patterning session isn't available, identify a single specific task to engage with — not a vague intention ("work on the project") but a concrete action ("write the opening sentence of section 3").
2. Direct attention fully to that task. The specificity is the mechanism: the more concrete the goal, the harder the prefrontal cortex engages and the more cleanly the DMN is suppressed.
3. If the anxiety resurfaces during the task, narrow the goal further. "Finish the section" → "write the next sentence" → "type this one word."
4. Log the underlying disturbance for later re-patterning. Goal engagement suppresses the output; re-patterning transforms the root.

## When to Use
In the moment of anxiety when a dedicated re-patterning session isn't possible — during work sessions, before presentations, mid-conversation. This is the fastest available lever. It doesn't fix the underlying pattern but occupies the neural circuitry that would otherwise run the anxiety loop, buying time and function until deeper work is available.

**Relationship to re-patterning**: Complementary, not competing. Consciousness Re-Patterning works at the root (transforming stored patterns that activate the DMN's threat-scanning). DMN Goal Engagement works at the output (suppressing the DMN's expression in real time). Both lead to the same destination from different directions.

## Key Insight
> "The brain defaults to rumination, not rest. Task engagement is anxiety suppression's fastest lever."
