# Observer Redirect

> When self-evaluation activates, shift identification from "I am the emotion/performance" to "I am watching it" — then redirect attention to what the work is doing, not how you're doing.

## Protocol

**When an evaluative spike hits mid-session:**

1. **Shift to observer**: "Something is activating." → "There is [frustration/anxiety/doubt] moving through right now." Notice the part watching the emotion is not itself emotionally activated — that's the observer. It's already neutral.

2. **Redirect to process**: Return to your pre-written process cue — the specific HOW instruction you set before the session ("follow the sentence," "read the test output," "stay with this section"). Don't argue with the evaluative voice. Just redirect. The cue is a reflex trigger, not a decision.

3. **Redirect to effect**: If the evaluative voice persists, shift attention to the external output: "What is the work doing right now?" (Writer: what the reader would experience. Coder: what the test tells you. Designer: what the user's path is.) This bypasses self-monitoring circuits structurally.

**Pre-session setup (takes 2 minutes):**
- Write 1–3 process cues for the session: specific, action-focused HOW instructions, not outcome goals.
- Assume the observer position for 60 seconds: notice whatever emotional weather is present, acknowledge without resistance. Begin from that vantage point.

## When to Use

Any time the evaluative voice is active — high-stakes creative work, performance situations, anything where "how am I doing?" is running in the background. The pre-session setup reduces the number of mid-session redirects needed. Establish process cues before the session so the redirect is automatic when the evaluative voice appears.

## Key Insight

> "The inner critic can only evaluate results. Give it a process and it has nothing to work with. Point attention at the work's effect and there's no 'self' left to monitor."
