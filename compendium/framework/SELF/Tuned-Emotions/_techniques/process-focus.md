# Process Focus

> Before any session, define 1–3 process cues describing HOW to do the work — when the evaluative voice activates, redirect to the cue immediately.

## Mechanism
Self-monitoring activates when attention drifts to outcomes — how this will be judged, how it compares to prior performance, whether it is good enough. When attention is fully occupied by *process* — the specific next action, the physical mechanics, the immediate procedural step — there is no "result" for the monitor to assess. Process cues structurally crowd out the inner critic by occupying the prefrontal cortex with specific, goal-directed content. The critic doesn't get suppressed by willpower — it's left with nothing to evaluate.

## Protocol

1. Before any performance-oriented session, identify 1–3 process cues: specific, action-focused instructions describing HOW to do the next thing (not what result to achieve). Examples:
   - Writer: "Follow the sentence." (Not: "make this good.")
   - Programmer: "Run the test, read the error, fix one thing." (Not: "get this working.")
   - Presenter: "Look at this person, finish this sentence." (Not: "be impressive.")
   - Musician: "Hear the next note before you play it." (Not: "play perfectly.")
2. Write the cue(s) somewhere visible before starting.
3. When the evaluative voice activates ("This is bad," "I'm not good at this"), do not argue with it. Return immediately to the process cue. The cue is the redirect — use it as a reflex, not a decision.

## When to Use
Before any session where self-consciousness has historically been a problem — high-stakes performance, creative work, public-facing output. Also effective as a general practice in any session, since some degree of self-monitoring is always present. The cue system becomes more powerful the more consistently it's used; the redirect becomes automatic.

## Key Insight
> "The inner critic can only evaluate results. Give it a process and it has nothing to work with."
