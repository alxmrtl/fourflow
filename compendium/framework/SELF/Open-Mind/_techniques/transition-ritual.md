# Transition Ritual

> Moving from one context to another without a deliberate close leaves the previous context running in the background — you arrive in the new one already partially occupied.

## Protocol

1. **Close the previous context** — before switching, take 60–90 seconds to complete a brief written handoff note for the prior task. The note contains three things only: (a) where you stopped, in one sentence; (b) the single next action when you return; (c) anything that must not be forgotten. This is not a summary — it is a "safe state" save, like saving a file before closing the application.
2. **Declare the close** — say or write a brief statement that the prior context is done for now. "Email project is parked. Next action: draft the reply to [name] before Thursday." Explicit declaration has been shown to reduce intrusive thought frequency more than implicit closure. This step takes 5 seconds.
3. **Insert a physical transition** — stand up, move to a different physical location if possible (even briefly), change your posture. The physical movement provides a proprioceptive marker that separates the two contexts. If you cannot move, do 3 slow neck rolls and reset your posture. The body needs a context boundary as much as the mind does.
4. **Set the incoming context** — before sitting to the new task, name it explicitly: "I am now working on [X]." If using a physical anchor cue (see physical-anchor-cue.md), perform it now. This is the final step — it activates the new context from a clean starting state rather than a residue-loaded one.
5. **Allow 2–3 minutes of initial difficulty** — even with a clean transition, the new context's neural configuration takes time to establish. Do not interpret slow initial engagement as a sign the ritual failed. Stay with the task; the residue dissipates within the first 5 minutes of active engagement.

## When to Use

- Anytime you are switching between meaningfully different work contexts — especially from reactive work (email, messages, meetings) to generative work (writing, building, designing).
- After any interruption longer than 2–3 minutes — the prior task is at risk of leaving residue every time focus was broken.
- At the end of the workday before transitioning to personal life — the transition ritual applies to the largest context switch of the day.

## Key Insight

> "A context doesn't end when you stop doing it — it ends when you close it. Everything else is just a pause."
