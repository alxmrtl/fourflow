# DMN Weed Model

> Understand possessions as weeds in the Default Mode Network garden — occupying idle-state processing that would otherwise produce creative synthesis, narrative coherence, and intrinsic motivation.

## Protocol

1. Identify times when creative insight or motivation typically surfaces for you (walking, showering, transitional moments between tasks).
2. Notice whether those moments have become less generative recently. If yes, ask: "What am I carrying in background attention that wasn't there before?"
3. Survey recent accumulations (new purchases, new commitments, new subscriptions). Each is a potential weed.
4. Use the [[attention-math-framework]] to quantify which items are highest-cost.
5. Release systematically. The test for the garden's health: do insights and motivation return in idle moments? If yes, the clearing is working.

## When to Use
When creative output feels flat even though focused work sessions seem productive. When motivation during "down time" is low. When transitions between tasks feel heavy. These are signs that the DMN garden is choked — the problem is not in the sessions themselves but in what's running between them.

## Key Insight
> "Possessions possess the mind — the clutter isn't only in the room, it's in the background processor running 24/7."
