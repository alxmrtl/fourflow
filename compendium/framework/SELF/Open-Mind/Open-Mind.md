# Open Mind

> The cognitive clarity layer — reducing mental noise so the mind can fully absorb the present task without interference from accumulated load, attention residue, or fixed beliefs.

## Why It Enables Flow

Flow requires the mind to be fully available to what's happening now. But the modern default is fragmentation: attention split across open loops, task-switching that leaves residue on everything touched, dopamine systems desensitized by overstimulation, and rigid beliefs that turn new inputs into threats. Each of these conditions taxes working memory, reduces signal clarity, and makes deep absorption impossible.

The failure mode is Attention Deficit Trait — Hallowell's term for acquired cognitive overwhelm. Unlike ADD, it has no genetic component; it's a learned response to chronic information overload. The mind loses resolution, toggles between tasks without finishing any, and operates in a persistent state of low-level panic. Flow becomes structurally inaccessible — not because the person can't focus, but because the conditions that allow focus have been eroded.
