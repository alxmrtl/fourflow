# Continuous Partial Attention

> The modern default of never giving full attention to anything — a perpetual alert state that blocks deep absorption and gradually erodes the capacity for it.

## What It Is

Continuous partial attention (CPA) was coined by tech strategist Linda Stone to describe the habitual state of scanning multiple streams simultaneously — monitoring email, social feeds, notifications, and conversations at once. It is distinct from multitasking: CPA is not an attempt to do multiple things but a perpetual alert state, always available, never fully present.

The state is self-reinforcing because it activates the dopamine novelty loop. Each new notification delivers a micro-reward. Over time, the brain habituates to this pace and begins to experience focused, single-task work as boring or uncomfortable.

## Why It Blocks Flow

Flow requires the attention to narrow completely onto the present task. CPA is the structural opposite: attention is kept wide, shallow, and scannable. The arousal level is high but unfocused — which means the brain is technically "engaged" but not in the coherent, absorbed way that produces flow.

The deeper problem is that CPA erodes the capacity for deep focus over time. The longer someone operates in CPA, the more uncomfortable single-task absorption becomes. The neural pathways for sustained focus weaken from disuse — it's not just a momentary distraction problem but a trained degradation of the focusing mechanism itself.

## Application

The key intervention is reducing the number of streams demanding attention — closing tabs, silencing notifications, and creating clear windows of single-stream focus. The goal is not to never scan, but to give scanning a designated place so that deep work windows are genuinely protected.
