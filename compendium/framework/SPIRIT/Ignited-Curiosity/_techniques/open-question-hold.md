# Open Question Hold

> Enter work with an answer to defend and you'll miss everything. Enter with an open question and work becomes a search.

## Protocol
1. Before beginning work, pause for 60 seconds. Do not open files, start typing, or review previous work yet.
2. Ask: "What is the genuine open question in this work right now?" Not the task ("write section 3"), but the actual question the work is trying to answer ("What is the real reason X doesn't work?"). Write it down.
3. Evaluate whether the question is genuinely open. A test: can you honestly imagine discovering an answer that would surprise you? If not, the question is rhetorical — you already have the answer. Find a more open version.
4. Write the question at the top of your working document or on a visible sticky note. You are now holding the question, not answering it.
5. Begin work while holding the question. The question functions as a perceptual filter — information relevant to it will become more visible; you'll notice things you would otherwise skip.
6. At session end: write one sentence about what the session revealed about the question. Not the answer — what you learned about the question. This is often more valuable than the answer.

## When to Use
- At the start of any creative, intellectual, or problem-solving session — making it a pre-session ritual builds the habit of entering work in seeking mode
- When work feels like going through motions — the question hold shifts mode from execution to discovery, often restoring engagement
- When stuck: the stuckness is usually in proving mode (defending an approach). Reformulate the core challenge as a genuine open question — the seeking mode often finds lateral moves that proving mode blocked

## Key Insight
> "A genuine question is a commitment to stay uncertain long enough to find something real."
