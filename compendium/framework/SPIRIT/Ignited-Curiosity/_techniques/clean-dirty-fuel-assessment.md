# Clean/Dirty Fuel Assessment

> Three diagnostic methods for identifying whether motivation is sourced in authentic interest (clean fuel) or fear/inadequacy (dirty fuel) — the distinction that determines whether flow is accessible.

## Mechanism
Dirty fuel (fear, proving worth, inadequacy) keeps the protective ego active — monitoring for failure, managing identity stakes — which fragments the attention flow requires. Clean fuel (curiosity, genuine interest, love, contribution) quiets the ego because there's nothing to prove; full attention becomes available. Same activity, radically different fuel, radically different outcome.

## Protocol

**Diagnostic 1 — Pushed vs. Pulled:**
- Pushed: "I have to / I should / or else..." → dirty fuel signal
- Pulled: "I want to / I'm curious about / I'm drawn toward..." → clean fuel signal

**Diagnostic 2 — Why-Deepening:**
1. Ask "Why do I want this?" and keep asking until you hit bedrock
2. Bedrock in fear, inadequacy, or "so I can feel OK about myself" = dirty fuel
3. Bedrock in genuine interest, contribution, or "because I care about this" = clean fuel

**Diagnostic 3 — Process vs. Completion Orientation:**
- Dirty fuel: dread before starting, relief when finished, focus on getting to done
- Clean fuel: engagement during the process, satisfaction from the work itself, time distortion

**The Anonymity Test (most revealing):**
Ask: "Would I want to do this if no one ever knew? If it led nowhere? If there was no status, money, or recognition?" What survives this test is closer to clean fuel than almost anything else.

**When dirty fuel is identified:**
Accept it fully — denial creates inefficiency and blocks self-awareness. From that clarity, explore the underlying pattern: "What am I really trying to get from this?" Working on the pattern (through re-patterning or inner work) dissolves the need for the achievement to provide what the pattern promised. What remains is whatever genuine interest existed.

## When to Use
Before any significant project or goal. When motivation is grinding or effortful. When arrival feels like the only relief. When you're unsure whether persistence is resilience or compulsion.

## Key Insight
> "Clean fuel makes the journey rewarding. Dirty fuel makes arrival the only relief."
