# Grounding Values

> The internal compass — the core principles that function as navigation when external direction is absent. When work aligns with values, there is an intrinsic energy available that no external reward can replicate. When work violates them, there is friction so fundamental that flow becomes structurally blocked.

## Why It Enables Flow

Values are not aspirations — they are revealed preferences: what you actually choose when it costs something. Work that is values-aligned carries an intrinsic motivational charge that extrinsic rewards can't fully replicate. The alignment itself is energizing; the misalignment is draining, regardless of performance metrics.

The failure mode is the values-behavior gap: the divergence between stated values and actual daily choices. This gap creates a low-level cognitive dissonance that drains energy constantly, operating below the level of conscious awareness. You can be technically productive while deeply misaligned — the system works, but the signal is corrupt. Sustained flow in this state is impossible because the values layer keeps generating friction against the work.
