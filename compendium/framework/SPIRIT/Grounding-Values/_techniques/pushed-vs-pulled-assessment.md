# Pushed vs. Pulled Assessment

> A quick diagnostic for motivation source — distinguishing fear/obligation-driven action (pushed) from interest/meaning-drawn action (pulled).

## Protocol

For any significant project or goal, ask: **"Am I being pushed or pulled here?"**

**Pushed signals:**
- Action driven by fear, obligation, or escape
- Internal language: "I have to / I should / or else..."
- Motivation requires external pressure or willpower
- Feels like forcing, efforting, grinding

**Pulled signals:**
- Action drawn by interest, curiosity, or meaning
- Internal language: "I want to / I'm curious about / I want to see where this goes..."
- Motivation arises naturally
- Feels like engagement even when the work is hard

**If pushed:** Explore what pattern is driving it. Is this genuine necessity (something must be done regardless) or avoidance of something you fear? Necessary pushed work can be done consciously; pattern-driven pushing can be transformed.

**If pulled:** Lean in. This is where clean fuel lives and where flow is most accessible.

**The nuance**: Not all pushed work is wrong — sometimes necessity requires it. But knowing the difference lets you understand where you're aligned and where you're overriding.

## When to Use
Before committing to a significant project or goal. When motivation is flagging mid-project (to diagnose whether it's resistance or genuine misalignment). When choosing between competing priorities.

## Key Insight
> "You can't flow toward something you're simultaneously running from."
