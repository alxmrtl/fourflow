# Daily Values Touchpoint

> Thirty seconds of contact with what matters most — because values erode through neglect, not attack.

## Protocol
1. Choose one core value to anchor (start with one; the touchpoint dilutes if you're cycling through a list).
2. Design a 20–60 second touchpoint ritual that creates genuine contact — not just recitation. Options:
   - A physical object associated with the value (hold it, feel it)
   - A single written sentence expressing the value as a present-tense truth ("Today I act from X")
   - One breath taken while silently naming the value
   - A brief behavioral act that expresses the value (e.g., if the value is generosity — send one message of genuine acknowledgment)
3. Anchor the touchpoint to an existing daily habit (morning coffee, before first task, post-shower). Do not make it a standalone event requiring scheduling.
4. Keep it fixed. Do not vary the form day-to-day — consistency of form is what produces encoding. The repetition is the mechanism.
5. After 30 days: notice whether value-aligned decisions feel more automatic — less deliberate, less costly. This is the signal that encoding has occurred.

## When to Use
- As a permanent daily practice, not a response to drift (prevention, not correction)
- When beginning a new phase of work in which a specific value is particularly load-bearing — intensify the touchpoint for that period
- When you notice a value becoming abstract rather than lived — the touchpoint recorporealizes it

## Key Insight
> "What you touch daily stays alive. What you only think about becomes a belief about yourself — not a force in your life."
