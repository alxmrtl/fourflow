# Values as Creative Constraint

> Use a core value as the creative brief — the narrower the constraint, the more genuine the solution.

## Mechanism
Constraints increase creative output — this is one of the most replicated findings in creativity research. The mechanism is not inspiration; it's elimination. When the solution space is unconstrained, the mind defaults to familiar templates and borrowed forms. When a real constraint is applied, it rules out the default answers and forces the cognitive search deeper. The result is more original because the obvious options are off the table.

Values-as-constraints work through an additional mechanism: authenticity pressure. When the constraint is a core value ("how would someone with radical honesty approach this?"), the solution must be genuinely expressive of something real about you — not just clever or technically correct. This creates alignment between the act of creation and the identity of the creator. When those two things are aligned, the work draws on more of you. It's the difference between solving a problem and expressing a truth.

This technique also resolves a common creative paralysis: too much optionality. "I could make this in a hundred ways" generates decision fatigue and option anxiety. A values-based constraint eliminates the options that don't fit and leaves only the ones that are authentically available to you. The constraint is not limiting you — it's finding you.

## Protocol
1. Name the creative challenge clearly: what are you making or deciding?
2. Select one core value that feels relevant to the work — not necessarily the most comfortable one. Productive constraints often carry some friction.
3. Convert the value into a constraint question: "How would someone who holds [value] approach this?" or "What's the [value]-native solution here?" Write this question down.
4. Generate 3–5 solutions using only that constraint as your filter. Actively disqualify options that violate the value — even if they're technically good. The disqualification is part of the process.
5. Evaluate the surviving solutions: which one is most fully expressive of the value, not just compliant with it? This is the direction to pursue.

## When to Use
- When facing a creative decision with too many viable options (values-as-constraint narrows to what's genuinely yours)
- When work feels technically competent but hollow — a sign the value layer is absent
- When iterating on a project that has drifted from its original intent — reapplying the founding value often restores direction

## Key Insight
> "The right constraint doesn't limit what you can make — it reveals what only you can make."
