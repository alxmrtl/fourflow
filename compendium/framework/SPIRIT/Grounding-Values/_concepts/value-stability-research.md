# Value Stability Research

> Decades of longitudinal research (Rokeach, Schwartz) show that core values remain stable across the lifespan — but only when they are actively maintained. Under social pressure, peripheral values drift and eventually pull core values with them.

## What It Is

The empirical study of value stability over time begins with Milton Rokeach's foundational work in *The Nature of Human Values* (1973), in which he distinguished between terminal values (desired end-states, such as freedom, equality, wisdom) and instrumental values (preferred modes of behavior, such as honesty, ambition, courage). Rokeach's longitudinal research showed that terminal values are highly stable across the adult lifespan, while instrumental values show more susceptibility to social influence and situational pressure.

Shalom Schwartz's cross-cultural values research, conducted across more than 80 countries over three decades, refined this picture considerably. Schwartz identified a universal set of ten basic value types organized into a circular motivational structure: values in adjacent sectors of the circle are compatible; values in opposing sectors are in tension. His research confirmed that value priorities established in early adulthood show remarkable stability, with the major exception of autonomy values (which increase with age) and benevolence values (which peak in middle adulthood). Critically, Schwartz distinguished between "centrality" (how important a value is to the individual) and "accessibility" (how easily it comes to mind in daily decisions). High-centrality values can have low accessibility if they are not regularly activated — and inaccessible values cannot guide behavior even when they are genuinely held.

More recent longitudinal work (Vecchione, Alessandri, Barbaranelli, Caprara) tracking value priorities across 10–15 year periods found that while rank order of core values is stable, the experiential connection to those values — the felt sense of living by them — erodes under sustained social pressure toward conformity or under conditions of chronic stress.

## Why It Matters for Grounding Values

The constant quality serves two functions that the stability research illuminates. First, it counteracts the accessibility problem: even stable values can become functionally inert if they are not regularly activated. Daily values contact (through review, reflection, or values-anchored decisions) maintains accessibility, which is the mechanism by which stable values actually influence behavior. Second, it creates a protective layer against peripheral value drift — the process by which small, situationally-motivated compromises at the edges of the values system gradually pull central values toward conformity.

The Schwartz finding about tension between adjacent values is also operationally important: values conflicts (between, say, achievement and benevolence) are not failures of character but structural features of the values system. The constant quality does not eliminate tension — it ensures that core values remain legible when tension is navigated, rather than being suppressed below awareness to reduce the discomfort of conflict.

## Application

- Distinguish between core values (terminal, high-centrality) and peripheral values (instrumental, more situationally variable) — maintenance practices should protect the core
- Schedule regular values contact to maintain accessibility, not just stability — held but inaccessible values do not guide behavior
- Treat peripheral value drift as an early warning signal for core value erosion — address small compromises early rather than normalizing them
- When experiencing values conflict, use the Schwartz circular model to identify which tension type is active — compatibility tension (adjacent) vs. opposition tension (opposite sectors) require different navigation
- After high-pressure periods (intense work, major life transitions), conduct an explicit values audit — these periods generate the most accessibility erosion
