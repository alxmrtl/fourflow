# Values as Motivators

> Norman Feather and Shalom Schwartz established that values are not just moral guardrails — they function as motivational goals that direct attention, generate energy, and select action. Generative values practice activates this motivational function deliberately.

## What It Is

The motivational theory of values, developed most rigorously by Norman Feather and Shalom Schwartz, establishes that values function as trans-situational goals — abstract desiderata that guide evaluation and action across contexts, not just in morally charged situations. Feather's expectancy-value framework (building on Atkinson's achievement motivation theory) demonstrated that the valence of an action — how motivating it feels — is a function of both the expected probability of success and the degree to which success would express a held value. This means that values are not passive — they actively shape the motivational landscape, determining what opportunities feel worth pursuing.

Schwartz's extension added crucial structural detail. His theory of basic human values proposes that values function as motivational goals organized by their underlying motivational type: power, achievement, hedonism, stimulation, self-direction, universalism, benevolence, tradition, conformity, and security. Each type describes a category of motivation that values in that sector activate. Values do not merely judge action after the fact — they generate direction before action begins. When a value is accessible (active in working memory), it functions like a motivated goal state, biasing attention toward value-relevant information, increasing effort toward value-expressive actions, and generating positive affect in anticipation of value expression.

The distinction between values-as-constraints and values-as-motivators is empirically important. Research by Bardi and Schwartz (2003) showed that people high on specific value types spend more time in behaviors that express those values — not because they are more disciplined, but because those behaviors feel intrinsically rewarding. Values motivate not by threatening punishment for violation but by generating pull toward expression.

## Why It Matters for Grounding Values

Generative practice activates the motivational function of values — the pull rather than the guardrail. Most values-oriented practices in self-help and organizational culture use values as constraints: "we don't do this," "that's not who we are." This is the lowest-energy use of the values system. Feather and Schwartz's research identifies a far more powerful use: making values active as motivational goals that generate approach energy toward expression.

The Access Your Values technique in Grounding Values works this way — it is not a review of what to avoid but an activation of what to move toward. When values are primed at the start of a work session, Schwartz's accessibility mechanism ensures that work consistent with those values generates more intrinsic motivation than it would if the values were dormant. The generative quality is, in this sense, a technique for harvesting the motivational energy that values contain but that remain latent without deliberate activation.

## Application

- At the start of significant work sessions, prime the values most relevant to that work — explicitly naming them activates the Schwartz accessibility mechanism
- When motivation is low, diagnose whether the work is values-expressive (Feather's framework): low motivation is often a signal that a values connection has been obscured, not that the work is wrong
- Design tasks to make values-expression visible within them — the more clearly a task expresses a held value, the more intrinsic motivation it generates
- Use the Schwartz value types to identify your highest-energy motivational sectors — these are where values activation produces the greatest motivational return
- Distinguish between approach values (what you move toward) and avoidance-framed values (what you move away from) — reformulating avoidance values as approach values accesses the generative function
