# Responsibility Without Evaluation Pressure

Separate caring about the work from needing to be seen performing it — so the work gets your full attention instead of sharing it with the performance of the work.

## Protocol

1. Before a session, ask: "Am I doing this because I care about it, or because I'm being graded on it?" If the second, identify specifically where the evaluation is coming from — which person, deadline, or comparison.
2. Locate the evaluation source and reduce it structurally if possible: share less about the work in progress, remove visible metrics that activate comparison, work in sessions without accountability to others.
3. When evaluation pressure activates mid-session — noticing yourself performing for an imagined observer — name it: "That's the evaluation layer." Then redirect attention to the work itself.
4. Practice [[act-from-worth]]: take action from the felt position of already being adequate — the work as expression, not as proof.
5. Use [[unconditional-self-worth-meditation]] as a session entry practice when evaluation pressure is high.

*Deep work and performance are structurally incompatible. The more real the evaluation feels, the less available the mind is for genuine engagement.*

## When to Use

- Before sessions where judgment from others (real or imagined) has been splitting attention
- When perfectionism is slowing execution — usually evaluation pressure in disguise
- When work feels like an audition rather than a creative or problem-solving act

## Key Insight

> "When your worth is being evaluated, your attention goes to the evaluation. The work gets what's left."
