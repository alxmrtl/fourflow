# Reframe Via Arc

> Stuckness is a stage, not a verdict — name the stage and the story reopens.

## Protocol
1. Name the stuckness explicitly in one sentence: "I am stuck because ___."
2. Map it to an arc stage. Ask: "In a story about someone doing what I'm doing, which stage does this moment belong to?" (Ordeal is the most common; Refusal and Threshold crossings also generate stuckness.)
3. State the reframe aloud or in writing: "This isn't failure — this is the [stage name]. The arc requires this."
4. Name what the arc says comes next — not a plan, just the narrative expectation (e.g., "After the ordeal, the reward. Something will shift.").
5. Return to the work with the arc frame active.

## When to Use
- When stuckness has persisted for more than one session and has started to feel like identity ("I'm someone who can't do this")
- When present difficulty is being interpreted as evidence that the mission was wrong
- When motivation has collapsed and the work itself feels meaningless

## Key Insight
> "The ordeal doesn't mean you're failing the story — it means you're in it."
