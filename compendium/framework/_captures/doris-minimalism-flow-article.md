# Why Spartans, Samurai, and Billionaires All Did This One Thing

> **Author**: Rian Doris · **Type**: article

## Core Argument

Possessions directly suppress flow by raising baseline cognitive load — every object owned is a recurring attention subscription, not a one-time cost. Minimalism (intentional prioritization + removal of everything non-essential) is therefore a direct flow lever, not an aesthetic preference.
