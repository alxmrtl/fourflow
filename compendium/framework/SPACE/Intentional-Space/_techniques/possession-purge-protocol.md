# Possession Purge Protocol

> A one-day possession audit and elimination event — used to reach a target minimalism tier or reset after significant drift.

## Mechanism
Gradual decluttering sprawls into weeks, loses momentum, and rarely completes. A one-day event creates the finality that incremental decluttering never achieves — the constraint forces decision-making rather than indefinitely deferring it. Seeing all possessions aggregated in one room is often the most powerful moment: the visual quantity makes the abstract cognitive cost concrete, supplying the motivation to cut deeply. Items removed the same day are actually gone; items that "leave tomorrow" often don't leave at all.

## Protocol

1. **Commit to one full day** — not a gradual process. Block it in the calendar in advance.
2. **Remove everything from the target space** except bare essentials (bed, core work setup).
3. **Bring all possessions into one room** so the full inventory is visible at once. Seeing the aggregate quantity is part of the mechanism — it supplies the motivation to cut.
4. **Sort each item using the tier-appropriate filter**:
   - *Aggressive*: "Is this a tool that directly advances my craft?" No = cut.
   - *Tempered/Mild*: "What am I assuming I'll get from this — and is that worth the cost of ownership?" Anything requiring complex justification = cut.
5. **Target keeping roughly 15%** of what's currently owned (calibrate up for Tier 3).
6. **Assign the cut pile** to: donate/gift, trash/recycle, or sell. Assign same day.
7. **The cut pile leaves the same day** — don't let it linger. Arrange pickup, drop-off, or disposal before the day ends.

**Complexity of justification as a signal**: The more secondary considerations that arise for an item (sentimental value, future utility, resale value, "I might need this someday"), the higher the actual cognitive cost of the item. Complexity of justification is itself a measure of possession-weight.

## When to Use
As an initial calibration when committing to environmental simplicity for the first time. As a reset when possession load has drifted significantly above target tier. The annual audit handles minor drift; the purge protocol handles major drift or initial setup.

## Key Insight
> "The one-day constraint is the mechanism — it forces a decision event that gradual decluttering permanently defers."
