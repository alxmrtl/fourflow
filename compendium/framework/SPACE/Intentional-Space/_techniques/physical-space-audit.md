# Physical Space Audit

> Every object in your workspace is a claim on your attention — the audit removes the ones that haven't earned it.

## Protocol
1. Clear the workspace completely — move everything off the surface and out of the immediate visual field. Begin from zero rather than subtracting from the existing arrangement.
2. Return only what you will actively use in the current work session. Hold each item for one second before returning it: "Do I need this today?" If no, it does not return to the surface.
3. For each item you consider returning, apply the open-loop test: does this object have an unresolved task attached to it? A book you haven't read counts. A device with unread messages counts. A project folder for a project not active today counts. Items that pass the open-loop test go into a designated "not today" location — out of the primary visual field.
4. Remove all screens displaying content unrelated to the current session from the visual field. If a second monitor is not in use, turn it off or reposition it. Screensavers showing email counts are open-loop generators.
5. Conduct the audit at session start, not at the end of the previous session. Starting from a cleared surface is a state-entry cue as well as a cognitive overhead reduction.
6. Schedule a full audit (everything off, start from scratch) weekly. Daily: restore the cleared surface state before beginning work.

## When to Use
- At the start of any session requiring sustained deep focus
- When you notice attention pulling toward objects in the workspace rather than toward the task
- Before beginning a new project phase — clearing the space clears the context from the previous phase

## Key Insight
> The objects you aren't consciously looking at are still costing you attention — the audit makes the bill visible.
