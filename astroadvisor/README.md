# AstroAdvisor

A wisdom system that translates astrological newsletters into personal consciousness guidance for developing FourFlow.

## Purpose

This isn't astrology-as-entertainment. **It's astrology-as-mirror** for consciousness work.

**The bridge**: Cosmic energy patterns → Personal consciousness alignment → Manifested outcomes in FourFlow

## Quick Start

### Ingest a Newsletter
```
/astroadvisor

[Paste newsletter URL, text, or transcript]
```

I'll archive it, extract key themes, and update the current landscape.

### Get Guidance
```
/astroadvisor

[Ask your question about energy, timing, consciousness, or project alignment]
```

I'll read the archive, translate archetypes into YOUR context, and provide actionable guidance.
