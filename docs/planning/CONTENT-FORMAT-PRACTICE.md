# FourFlow PRACTICE Content Format Template

## Standard Structure for Exercise & Implementation Content

### Header Section
```
# [PRACTICE NAME]: [Action-Oriented Subtitle]
*A FourFlow Implementation Exercise*

**Dimension**: [SELF/SPACE/SPIRIT/STORY]
**Key**: [Specific key from dimension]
**Type**: PRACTICE
**Duration**: [Time to complete]
**Difficulty**: [Beginner/Intermediate/Advanced]
**Materials**: [What's needed to complete]
```

### 1. Practice Overview & Purpose (100-150 words)
Clear statement of what the practice accomplishes and why it matters.

**Structure:**
- What this practice does
- Primary benefit/outcome
- How it triggers flow state
- Connection to FourFlow framework

### 2. Scientific Foundation (150-200 words)
Evidence-based rationale for the practice.

**Include:**
- Research supporting the approach
- Neurological/psychological mechanisms
- Expected outcomes based on studies
- Why this specific method works

### 3. Prerequisites & Preparation (100-150 words)
What's needed before starting the practice.

**Include:**
- Required mindset/skills
- Physical preparation
- Environmental setup
- Time commitment

### 4. Step-by-Step Instructions (400-600 words)
Detailed, actionable guidance for completing the practice.

**Structure:**
- Phase 1: Setup (2-3 steps)
- Phase 2: Core Practice (5-8 steps)
- Phase 3: Integration (2-3 steps)
- Clear timing for each phase
- What to expect at each stage

### 5. Troubleshooting Guide (200-300 words)
Common challenges and solutions.

**Include:**
- 3-5 typical obstacles
- Specific solutions for each
- How to know if you're on track
- When to adjust approach

### 6. Progression Pathway (150-200 words)
How to develop mastery over time.

**Include:**
- Beginner → Intermediate → Advanced variations
- Frequency recommendations
- Signs of progress
- Advanced applications

### 7. Integration & Tracking (100-150 words)
How to incorporate into daily life and measure progress.

**Include:**
- Daily/weekly application
- Success metrics
- Journal prompts
- Habit stacking opportunities

### 8. Related Practices (50-100 words)
Connections to other FourFlow exercises.

**Include:**
- Complementary practices
- Sequential recommendations
- Synergistic combinations

## Exercise Design Principles

### Actionability
- Every step must be concrete and doable
- Clear start and end points
- Measurable outcomes
- Immediate feedback loops

### Progressive Challenge
- Multiple difficulty levels
- 4% edge principle (just beyond comfort zone)
- Clear advancement pathway
- Skill building sequence

### Flow Optimization
- Clear goals
- Immediate feedback
- Balance challenge/skill
- Deep concentration
- Merge action/awareness

### Integration Focus
- Connect to daily life
- Habit formation support
- Long-term sustainability
- Compound benefits

---

## Quality Standards

### Clarity
- Step-by-step instructions
- No ambiguous language
- Visual descriptions when helpful
- Common mistakes highlighted

### Completeness
- All necessary information included
- Nothing left to interpretation
- Troubleshooting covered
- Progression pathway clear

### Effectiveness
- Evidence-based approach
- Tested methodology
- Realistic expectations
- Measurable outcomes

### Safety
- Physical/emotional safety considered
- Contraindications noted
- When to stop/adjust
- Professional guidance recommended when appropriate
