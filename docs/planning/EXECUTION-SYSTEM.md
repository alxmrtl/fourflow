# DAILY LEVER SYSTEM

## The Daily Protocol

### Pre-Session (5 min)
1. Open this document
2. Identify today's lever from weekly plan
3. Set a 60-90 min timer
4. Close everything except what you need

### Session (60-90 min)
1. Work on ONE lever only
2. No email, no Slack, no phone
3. When timer ends, you're done (or continue if in flow)

### Post-Session (5 min)
1. Mark lever as DONE or NOT DONE
2. If not done, what % complete?
3. Note any blockers
4. Prep tomorrow's lever

# ANTI-PROCRASTINATION DESIGN

## Why This System Works

1. **Decisions are pre-made** - You don't decide what to do, you execute what's queued
2. **Tasks are small enough** - Every lever is 30-90 min, never overwhelming
3. **Binary accountability** - DONE or NOT DONE. No wiggle room.
4. **Visible progress** - Check marks accumulate. Momentum builds.
5. **Compounding results** - Every lever connects to the goal. Nothing is busywork.

## When You Don't Feel Like It

1. Set timer for just 25 minutes (Pomodoro)
2. Tell yourself: "I only have to start"
3. After 25 min, decide: continue or stop?
4. Most of the time, you'll continue

## When You Miss a Day

1. Don't stack. Don't try to "make up" missed days.
2. Just do today's lever today.
3. Consistency > intensity. 80% of days > 50% of days at 150%.

## When You're Stuck

1. Is the lever too big? Break it down.
2. Is something unclear? That becomes today's lever: clarify.
3. Is fear stopping you? Do the scared thing anyway. 60 min won't kill you.
